# Chip-8 Emulator

Chip-8 emulator in C++.

## How to use

1. Clone the repository
```
git clone https://github.com/timothewt/Chip8.git
cd Chip8
```

2. 
```
mkdir build
cd build
cmake ..
make
```

3. Start the program with one of the available ROMs
```
./chip8 ../roms/tetris.ch8
```

## Dependencies

- SDL2
